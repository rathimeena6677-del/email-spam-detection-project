# email-spam-detection-project
This project implements an Email Spam Detection system using Machine Learning. It processes text data, extracts features using CountVectorizer, and applies the Naive Bayes algorithm to classify messages as spam or not spam. The model achieves high accuracy and provides real-time predictions with confidence scores.
